# 🤖 AI Chatbot using Transformers

##  Project Title
AI Chatbot using Transformer Model (DialoGPT)

---

##  Project Description

This project implements an intelligent conversational chatbot using a transformer-based language model. The chatbot is capable of generating human-like responses and maintaining conversational context using Microsoft's DialoGPT model from the Hugging Face Transformers library.

The system processes user input, maintains conversation history, and generates meaningful responses in real-time through a command-line interface.

---

##  Objectives

- Build a conversational AI chatbot
- Understand transformer-based NLP models
- Implement context-aware response generation
- Apply Hugging Face Transformers in real-world applications
- Develop an interactive chatbot system

---

##  Technologies Used

- Python
- PyTorch
- Hugging Face Transformers
- DialoGPT Model
- Natural Language Processing (NLP)

---

##  System Requirements

- Python 3.8 or higher
- Internet connection (for first model download)
- Minimum 4GB RAM recommended
